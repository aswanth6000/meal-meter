# Releases

This folder contains packaged releases of the Meal Meter Chrome extension.

## Available Releases

- **meal-meter-v1.0.zip** - Latest release (v1.0)

## Installation Instructions

1. Download the ZIP file for the version you want to install.
2. Extract the ZIP file to a folder on your computer.
3. Open Chrome and navigate to `chrome://extensions`
4. Enable **Developer mode** (toggle in the top-right corner)
5. Click **Load unpacked**
6. Select the extracted folder (not the ZIP file)
